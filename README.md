Repositório onde as resoluções em código feitas para a disciplina de Modelagem Matemática serão depositadas.

As resoluções serão feitas em Python e em Lua.

Python requer a biblioteca matplotlib instalada na máquina através do pip ou do python3 para a plotagem dos gráficos.

Lua requer a biblioteca plotly baixada na máquina através do luarocks para a plotagem dos gráficos.
